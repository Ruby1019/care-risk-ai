# CareRisk AI

CareRisk AI is an AI-assisted healthcare process risk dashboard designed to analyze patient process event logs, identify workflow bottlenecks, calculate risk scores, and generate improvement suggestions for healthcare process management.

## Project Motivation

Healthcare workflows often involve multiple departments, waiting periods, and handoff points. Delays, missing steps, or abnormal process patterns may increase operational risk and reduce service quality.

This project aims to support healthcare process improvement by combining process analysis, risk scoring, dashboard visualization, and AI-assisted decision support.

## Key Features

- Upload healthcare process event logs
- Calculate activity duration and waiting time
- Identify delayed or abnormal cases
- Generate case-level risk scores
- Visualize healthcare workflow performance
- Provide AI-assisted improvement suggestions
